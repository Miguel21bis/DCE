DCE ScriptsMod

DCE (Dynamic Campaign Engine) is a dynamic campaign system for Digital Combat Simulator (DCS).

This repository contains the ScriptsMod component used by DCE campaigns.

DCE Components

DCE relies on three components:

1. DCE ScriptsMod (this repository)

  Provides the Lua scripts executed by DCS during missions.

2. DCE Manager

  The external application used to generate missions, process mission results, and advance the campaign.

3. DCE Campaign

  A campaign package created by the DCE community.

  Examples of DCE campaigns created by Cef include:

  Persian Gulf War
  Syria 1982
  Israeli Independence War
  Vietnam

Installation

  For most users, nothing needs to be downloaded from this repository.
  
  When downloading a DCE campaign created by Cef, everything required is already included:
  
  The campaign itself
  DCE ScriptsMod
  DCE Manager (DCE_Manager.exe)

  Simply follow the installation instructions provided with the campaign.


Why This Repository Exists

  This repository is mainly intended to distribute updates for the DCE ScriptsMod component.
  
  Likewise, the DCE Manager repository exists primarily to distribute updates for DCE_Manager.exe.
  
  Most users will obtain all three DCE components directly from their chosen campaign package.

Requirements
  Digital Combat Simulator (DCS)
  A compatible DCE campaign
